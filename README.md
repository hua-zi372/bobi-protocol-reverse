# Bobi APP Protocol Analysis & Reverse Engineering

> 针对波币(Bobi) APP 的网络通信协议逆向分析与自动化交互方案

## 📌 项目简介

本项目对波币APP的核心业务接口进行了协议层逆向分析，实现了从抓包分析、加密算法还原到自动化交互的完整链路。涵盖了移动端常见的协议对抗手段及其突破方案。

**声明：本项目仅用于安全研究与学习交流，请勿用于非法用途。**

---

## 🔍 逆向分析范围

### 1. 网络通信协议

| 分析项 | 技术栈 | 状态 |
|--------|--------|------|
| API Base URL 定位 | 抓包 + 反编译 | ✅ 已完成 |
| 请求签名机制 | HMAC / 自定义Hash | ✅ 已还原 |
| Token鉴权流程 | OAuth2.0变体 | ✅ 已还原 |
| WebSocket长连接 | 实时推送通道分析 | ✅ 已完成 |
| SSL Pinning绕过 | Frida / Xposed Hook | ✅ 已绕过 |

### 2. 加密算法还原

- **请求参数加密**：自定义AES变体 + 动态IV
- **签名算法**：多字段拼接 → HMAC-SHA256 → Base64
- **时间戳校验**：服务端时间同步 + 滑动窗口验证
- **设备指纹**：多维度采集 → MD5混合编码
- **Anti-Replay**：Nonce + Timestamp 双因子防重放

### 3. 关键业务接口

```
POST /api/v2/order/list      # 订单列表（含实时价格）
POST /api/v2/order/grab      # 抢单接口（核心）
POST /api/v2/pay/confirm      # 支付确认
POST /api/v2/user/bindcard    # 绑卡接口
WS   /ws/realtime             # 实时推送通道
```

> ⚠️ 具体请求/响应结构及加密细节不在本仓库公开

## 📊 协议对抗分析

### 检测到的防护措施

| 防护类型 | 技术手段 | 绕过难度 |
|----------|----------|----------|
| SSL Pinning | 证书绑定 + 公钥校验 | ⭐⭐ |
| Root检测 | 多路径探测 + Magisk检测 | ⭐⭐⭐ |
| 模拟器检测 | 硬件特征 + 传感器校验 | ⭐⭐ |
| 代理检测 | 系统代理 + VPN状态检测 | ⭐ |
| 请求频率限制 | 滑动窗口 + IP限流 | ⭐⭐⭐ |
| 设备指纹 | 多维度采集 + 行为分析 | ⭐⭐⭐⭐ |

### 绕过策略（概要）

1. **证书层**：运行时Hook替换证书校验链
2. **环境层**：虚拟化方案隐藏Root/模拟器特征
3. **协议层**：完整还原签名算法，构造合法请求
4. **频控层**：分布式请求 + 智能延迟策略

> 详细绕过实现属于核心成果，不对外公开
